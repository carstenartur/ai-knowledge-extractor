Plugin Documentation

Goals available for this plugin:

Goal Description
ai-knowledge:analyze Computes AI complexity metrics and trend report artifacts.
ai-knowledge:benchmark Compares extraction profiles and optional empirical fixtures against context budgets.
ai-knowledge:check Runs AI knowledge quality gates and fails the build on configured threshold violations.
ai-knowledge:generate Scans the repository and generates deterministic AI knowledge index artifacts.
ai-knowledge:help Displays plugin goal and parameter help for the AI Knowledge Maven Plugin.
ai-knowledge:optimize Detects knowledge smells and writes ranked optimization suggestions.

System Requirements

The following specifies the minimum requirements to run this Maven plugin:

Maven 2.0
JDK Default target for maven-compiler-plugin version 3.15.0
Memory No minimum requirement.
Disk Space No minimum requirement.
